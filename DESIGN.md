# BeiLuo Electronic Components Distributor - UI/UX Design Document

## 1. 设计概述

### 1.1 项目定位
BeiLuo是一个专业的电子元器件核心代理商B2B网站，面向全球电子工程师和采购人员，提供100+授权品牌的电子元器件产品、技术支持和解决方案。

### 1.2 设计风格
- **风格类型**: 现代极简主义 (Modern Minimalism)
- **设计语言**: 科技感、专业、可信赖
- **视觉密度**: 中等密度，信息清晰可读
- **内容优先**: 突出产品信息和技术内容

### 1.3 目标用户特征
- **职业**: 电子工程师、采购经理、技术决策者
- **年龄**: 25-55岁
- **使用场景**: 工作中查找产品规格、比较型号、获取技术支持
- **设备**: 桌面端为主(70%)，平板和移动端为辅

## 2. 设计系统

### 2.1 色彩系统

#### 主色调
```css
--color-primary: #0072ce;        /* 科技蓝 - 主品牌色 */
--color-primary-dark: #005ba3;   /* 深蓝 - 悬停状态 */
--color-primary-light: #e6f2fa;  /* 浅蓝 - 背景高亮 */
```

#### 辅助色
```css
--color-secondary: #f8f9fa;      /* 中性灰 - 背景 */
--color-secondary-dark: #e9ecef; /* 深灰 - 边框、分隔线 */
```

#### 功能色
```css
--color-accent: #ff6b35;         /* 橙色 - CTA按钮 */
--color-accent-hover: #e55a2b;   /* 深橙 - 悬停 */
--color-success: #28a745;        /* 绿色 - 有货状态 */
--color-warning: #ffc107;        /* 黄色 - 询价状态 */
--color-danger: #dc3545;         /* 红色 - 错误 */
```

#### 文字色
```css
--color-text-primary: #1a1a1a;   /* 主文字 - 标题 */
--color-text-secondary: #4a4a4a; /* 次要文字 - 正文 */
--color-text-tertiary: #6c757d;  /* 辅助文字 - 说明 */
--color-text-inverse: #ffffff;   /* 反色文字 - 深色背景 */
```

#### 色彩使用规则
- 主色调用于：Logo、导航激活状态、主要链接、品牌强调
- 辅助色用于：页面背景、卡片背景、分隔区域
- 功能色用于：CTA按钮、状态标签、反馈信息
- 文字色用于：内容层级区分

### 2.2 字体系统

#### 字体家族
```css
--font-primary: 'Inter', -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif;
--font-mono: 'JetBrains Mono', 'Fira Code', monospace;
```

#### 字体规模
| 层级 | 大小 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| H1 | 48px | 700 | 1.2 | 页面主标题 |
| H2 | 36px | 600 | 1.3 | 区块标题 |
| H3 | 28px | 600 | 1.3 | 子区块标题 |
| H4 | 22px | 600 | 1.4 | 卡片标题 |
| H5 | 18px | 600 | 1.4 | 小标题 |
| Body Large | 18px | 400 | 1.6 | 重要正文 |
| Body | 16px | 400 | 1.6 | 标准正文 |
| Body Small | 14px | 400 | 1.5 | 辅助文字 |
| Caption | 12px | 400 | 1.4 | 标签、注释 |

#### 字体使用规则
- 标题使用Inter字体，字重600-700
- 正文使用Inter字体，字重400，行高1.6
- 代码、型号使用等宽字体JetBrains Mono
- 中文内容使用系统默认中文字体

### 2.3 间距系统

#### 基础单位
```css
--space-unit: 8px;
```

#### 间距规模
| Token | 值 | 用途 |
|-------|-----|------|
| --space-xs | 4px | 图标内边距、紧凑间距 |
| --space-sm | 8px | 小组件间距 |
| --space-md | 16px | 标准组件间距 |
| --space-lg | 24px | 区块内边距 |
| --space-xl | 32px | 大区块间距 |
| --space-2xl | 48px | 页面级间距 |
| --space-3xl | 64px | Hero区域间距 |
| --space-4xl | 96px | 大区块分隔 |

#### 间距使用规则
- 组件内部使用--space-sm到--space-md
- 区块之间使用--space-lg到--space-xl
- 页面大区块使用--space-2xl到--space-4xl
- 保持8px的倍数节奏

### 2.4 圆角系统
```css
--radius-sm: 4px;    /* 小元素：标签、按钮 */
--radius-md: 8px;    /* 标准元素：输入框、卡片 */
--radius-lg: 12px;   /* 大元素：卡片、模态框 */
--radius-xl: 16px;   /* 特殊元素：Hero区域 */
--radius-full: 9999px; /* 圆形：头像、徽章 */
```

### 2.5 阴影系统
```css
--shadow-sm: 0 1px 2px rgba(0, 0, 0, 0.05);
--shadow-md: 0 4px 6px rgba(0, 0, 0, 0.07);
--shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1);
--shadow-xl: 0 20px 25px rgba(0, 0, 0, 0.15);
--shadow-glow: 0 0 20px rgba(0, 114, 206, 0.3); /* 主色光晕 */
```

## 4. 组件设计

### 4.1 按钮 (Button)

#### 主按钮 (Primary)
- **背景**: --color-accent
- **文字**: --color-text-inverse

- **内边距**: 12px 24px
- **字体**: 16px, 字重600
- **悬停**: 背景变深，阴影增强
- **过渡**: 150ms ease-out

#### 次按钮 (Secondary)
- **背景**: transparent
- **边框**: 2px solid --color-primary
- **文字**: --color-primary
- **悬停**: 背景填充--color-primary-light

- **背景**: transparent
- **文字**: --color-primary
- **悬停**: 下划线出现

- **圆角**: --radius-lg (12px)
- **阴影**: --shadow-md
- **内边距**: --space-lg (24px)
- **悬停**: 上移4px，阴影变为--shadow-lg
- **过渡**: 200ms ease-out

#### 品牌卡片
- **背景**: white
- **圆角**: --radius-lg (12px)
- **边框**: 1px solid --color-secondary-dark
- **内边距**: --space-lg (24px)
- **悬停**: 边框变为主色，阴影出现

### 4.3 表格 (Table)

### 4.4 表单 (Form)

#### 输入框

### 4.5 导航 (Navigation)

## 6. 交互设计

### 6.1 全局交互

#### 页面加载
- 显示骨架屏或加载指示器
- 首屏内容优先加载
- 图片懒加载

#### 滚动行为
- 导航栏滚动后固定并添加阴影
- 平滑滚动到锚点
- 返回顶部按钮（滚动超过一屏后显示）

#### 链接行为
- 所有链接有悬停状态
- 外部链接新窗口打开
- 当前页面链接不跳转

### 6.2 组件交互

#### 按钮
- 悬停：背景色加深，轻微上移
- 点击：缩放0.98效果
- 禁用：透明度0.5，不可点击
- 加载：显示旋转指示器

#### 卡片
- 悬停：上移4px，阴影增强
- 过渡：200ms ease-out

#### 表格行
- 悬停：背景高亮
- 点击：跳转详情页

#### Tab切换
- 激活项：底部边框高亮
- 内容切换：淡入淡出

### 6.3 表单交互

#### 输入框
- 聚焦：边框变主色，光晕效果
- 错误：边框变红色，显示错误信息
- 成功：显示勾选图标

#### 下拉选择
- 点击：展开选项列表
- 选择：列表收起，显示选中项

#### 提交按钮
- 点击：禁用并显示加载状态
- 成功：显示成功消息
- 失败：显示错误消息，恢复可点击

## 7. 响应式设计

### 7.1 桌面端 (>= 1024px)
- 完整布局，所有功能可用
- 多列网格显示
- 侧边栏固定显示

### 7.2 平板端 (768px - 1023px)
- 网格列数减少
- 部分侧边栏收起或移至底部
- 导航栏保持固定

### 7.3 移动端 (< 768px)
- 单列布局
- 汉堡菜单导航
- 表格横向滚动
- 侧边栏移至内容底部
- 触摸目标最小44px

---

## 8. 可访问性设计

### 8.1 色彩对比
- 文字与背景对比度 >= 4.5:1
- 大文字对比度 >= 3:1
- 交互元素对比度 >= 3:1

### 8.2 键盘导航
- 所有交互元素可通过Tab键访问
- 焦点状态清晰可见
- 跳过导航链接

### 8.3 屏幕阅读器
- 所有图片有alt属性
- 表单元素有关联标签
- 使用语义化HTML标签
- ARIA标签用于复杂组件

### 8.4 动画
- 支持prefers-reduced-motion
- 动画时长不超过300ms
- 不闪烁或快速闪烁
